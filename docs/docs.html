DIP: Development In Progress


What is DIP?


DIP is a text editor designed for efficient programming in the fast-paced world of software development.


What makes DIP different?


Unlike other code editors, DIP is lightweight, completely customizable, allows for very easy creation of extensions, is low on CPU consumption, and has a minimal RAM footprint.


How does the styling work?


DIP utilizes a few basic .json files for the styling. These files reside in the users app data directory and the files controlling styling are highight.json and master.json. highlight.json controls the styling of individual widgets such as buttons, tooltips, scrollbars, and even the line numbers. It also controls the font used throughout the app, font sizes, and window placement. On the other hand, master.json controls the coloring for the syntax highlighting. In the future, we plan to add a GUI for modifying the apps styling and syntax highlighting. To learn more about the styling, please visit the styling section of the documentation.


How do I create an extension?


DIP extensions are written in Python and are very easy to create. This topic, however, is too large for the overview section of the documentation. For more information on creating extensions, please visit the extensions section.


What is DIP written in?


DIP is written in Python and utilizes the tkinter library for the GUI, pygments and tree-sitter for syntax highlighting, pyperclip for proper copy/paste functionality, platformdirs for help with file management, requests for dealing with servers and update detection, chlorophyll for help with syntax highlighting, and PyInstaller for creating executables.
